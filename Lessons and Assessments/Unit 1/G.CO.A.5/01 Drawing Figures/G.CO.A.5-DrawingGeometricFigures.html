Drawing Geometric Figures with GeoGebra

GeoGebra is an interactive tool for Geometry, Algebra, Statistics and Calculus. Learning how to use GeoGebra will be useful because it can be used on the computer, tablet, or phone. This lesson will demonstrate how to create points, lines, and geometric figures using technology.

Example 1.1 - Creating Points

  1. To create points using GeoGebra click on the “point tool” highlighted yellow in the video to the right.
  2. GeoGebra allows the user to create many types of points. For this lesson select the first option as seen in the video to the right.
  3. Next place the point anywhere on the graph. These points will automatically be labeled. The coordinates of each point will appear next to the graph. To rename or make adjustments double click on the coordinates.

Example 2.1: Creating Lines

  1. To set up the graph workspace in GeoGebra you can right click the mouse to bring up display options for both the graph grid and axes.
  2. To create a line using GeoGebra click on the “line tool” highlighted with a red circle in the demonstration to the right.
  3. Click on any points to create the line. In the example to the right, when the origin \((0,0)\) was clicked on, point \(A\) was created.
  4. Next \((3,3)\) was clicked on which created point \(B\) and a line connecting points \(A\) and \(B\). On the left side of the graph the coordinates for the two points along with the function will be displayed.

3. How do I graph a linear equation in GeoGebra?
Instruction Video Self-Check

Example 3.1: Graphing Linear Equations in GeoGebra

Graph \(3x + 2y = 10\)

  1. Write the equation in slope intercept form.

    \(3x + 2y = 10\)

    \(3x - 3x + 2y = 10 - 3x\)

    \((2y)/2 =(-3x + 10)/2\)

    \(y= -3/2 x + 5\)

  2. Identify the slope and \(y\)-intercept

    Slope = \(-3/2\)

    \(y\)-intercept = \(5\)

  3. Plot the \(y\)-intercept

  4. Use the slope to plot the second point

  5. Connect the two points to create a line.

Self-Check 3.1

Self-Check 3.1 - GeoGebra Solution
×

To change the equation into slope intercept form you must first subtract \(6\) on both sides of the equation. By doing this, you get \(2y = 2x – 6\). We are almost there. We now need to isolate \(y\) on the left side of the equation. We do that by dividing both sides by \(2\). Remember to divide everything on the right side by \(2\). This gives us \(y = x – 3\).

4. How do I create a geometric figure using GeoGebra?
Instruction Video Self-Check

Example 4.1: Creating a Polygon

  1. To create a polygon using GeoGebra click on the “polygon tool” and select the appropriate option. GeoGebra has the following options to choose from: polygon, regular polygon, rigid polygon, and vector polygon.
  2. Select the type of polygon and plot each vertice of the polygon. In the picture below notice that only \(2\) sides of the polygon are solid.
  3. To complete the polygon click on the first vertex to close the polygon and all sides will become solid.

Example 4.2: Creating a Circle

  1. To create a circle or ellipse using GeoGebra click on either the “circle tool” or “ellipse tool” and select the option needed.
  2. To create a circle with the center at \((1,2)\) and a radius of \(3\) select “circle with a center and radius”.
  3. To create a circle start by clicking on the center \((1,2)\) then click on any point \(3\) units away from the center. On the left the equation of the circle will be displayed.
