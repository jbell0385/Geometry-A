Performing a Sequence of Transformations

Objectives

I can use GeoGebra (technology) to perform a sequence of transformations.

  1. I can use GeoGebra to reflect a given image.
  2. I can use GeoGebra to rotate a given image.
  3. I can use GeoGebra to translate a given image.
  4. I can use GeoGebra to perform a sequence of transformations on a given image.

Reflection
×
A transformation across a line that produces a mirror image of the original (preimage) shape. The reflection is called the image of the original figure. The line is called a line of symmetry.
Close
Rotation
×
A shape has rotational symmetry when it can be rotated less than \(360Â°\) about a point, called the center or rotation, and appears to not change.
Close
Translation
×
Translation is a type of rigid transformation in which an object slides on a graph. It does not change its shape or size.

1. GeoGebra Reflections
Instruction Video Self-Check

In order to perform a reflection using GeoGebra perform the following:

  1. Select the "Reflect About Line" tool.
  2. Select the triangle you are reflecting.
  3. Select the line of reflection.
  4. The reflection should now appear.

2. GeoGebra Rotations
Instruction Video Self-Check

In order to perform a reflection using GeoGebra perform the following:

  1. Select the "Point" tool.
  2. Create a center of rotation point.
  3. Select the "Rotate Around Point" tool.
  4. Select the triangle.
  5. Select the center of rotation.
  6. Enter in a degree of rotation.

4. GeoGebra Sequence of Transformations
Instruction Video Self-Check

In order to perform a translation using GeoGebra perform the following:

  1. Create a Center of Rotation point.
  2. Rotate the \(\triangle ABC\) \(180^{\circ}\)
  3. Create a vector line.
  4. Translate \(\triangle A'B'C'\) \(\langle 0,2 \rangle\).
  5. Next, Create a line of reflection using the line tool.
  6. Reflect \(\triangle A'B'C'\) over the line of reflection.

Try It Out

In the GeoGebra below try the following:

  1. Rotate \(\triangle ABC\) \(180^{\circ}\) about \((1,2)\).
  2. Then, translate \(\triangle A'B'C'\) by \(\langle 2,0 \rangle\).
  3. Finally, reflect \(\triangle A''B''C''\) over the line \(x=2\).

\(\triangle A'''B'''C''' = r_{x=2} \circ T_{\langle 0,2 \rangle} \circ R_{(1,2),180^{\circ}}\)

Self-Check 4.1

What is the coordinate value of \(C'''\) if you do the following transformations to \(\triangle ABC\):

  1. Rotate \(\triangle ABC\) \(90^{\circ}\) about \((0,0)\).
  2. Reflect \(\triangle A'B'C'\) over the line \(y=0\).
  3. Translate \(\triangle A''B''C''\) by \(\langle 1,-1 \rangle\).

\(\triangle A'''B'''C''' = T_{\langle 1,-1 \rangle} \circ r_{y=0} \circ R_{(0,0),90^{\circ}}\)

Answer
Self-Check 4.1
×

The coordinate of \(C'''\) is \((0,-4)\).
