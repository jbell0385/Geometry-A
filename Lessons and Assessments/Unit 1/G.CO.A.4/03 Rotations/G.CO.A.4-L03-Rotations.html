1. Attributes of Rotation
Instruction Examples Video Self-Check

What Is a Rotation?

A rotation is turning an object around a center point.

  1. Rotations are one of the four transformations (translations, reflections, rotations, and dilations).
  2. Rotations are rigid, which means the image of the figure that is rotated is congruent to the pre-image.
  3. Rotations are rotated around a center, which is usually a coordinate (e.g. \((2,3)\))
  4. The amount of rotation is usually in degrees or radians.

Center of Rotation

A rotation needs a center of rotation that the figure is rotated around. Often the center of rotation is the origin \((0,0)\), but it can be any coordinate. If a center of rotation is not indicated, assume it is the origin.

Direction of Rotation

There are two possible directions of rotation:

  1. clockwise
  2. counter-clockwise

In math, if a direction is not indicated, then assume the direction of rotation is counter-clockwise.

Rotation Notation

The notation for a rotation is as follows: \[R_{(90^{\circ},(2,3))}(\triangle ABC)\]

  1. The \(R\) represents "Rotation"
  2. The \(90^{\circ}\) spot is your amount of rotation
  3. The \((2,3)\) spot is your center of rotation.
  4. The \(\triangle ABC\) spot is what you are rotating. It could be a point, line, or a figure.
  5. Note: If you see \(O\), that represents "origin", which is \((0,0)\).

2. Rotations Around the Origin - Counter-Clockwise
Instruction Examples Video Self-Check

When a coordinate or figure is rotated counter-clockwise around the origin, then the following is true:

  1. \(R_{(90^{\circ},O)}(x,y) = (-y,x)\)
  2. \(R_{(180^{\circ},O)}(x,y) = (-x,-y)\)
  3. \(R_{(270^{\circ},O)}(x,y) = (y,-x)\)
  4. \(R_{(360^{\circ},O)}(x,y) = (x,y)\)

Example 2.1 - Rotation by 90° - Counter-Clockwise

Rotate \(\triangle ABC\) counter-clockwise by \(90^{\circ}\) about the origin.

  1. Remember \(R_{(90^{\circ},O)}(x,y) = (-y,x)\).
  2. A: \(R_{(90^{\circ},O)}(1,2) = (-2,1)\).
  3. B: \(R_{(90^{\circ},O)}(1,3) = (-3,1)\).
  4. C: \(R_{(90^{\circ},O)}(-2,3) = (-3,-2)\).

Example 2.2 - Rotation by 180° - Counter-Clockwise

Rotate \(\triangle ABC\) counter-clockwise by \(180^{\circ}\) about the origin.

  1. Remember \(R_{(180^{\circ},O)}(x,y) = (-x,-y)\).
  2. A: \(R_{(180^{\circ},O)}(1,2) = (-1,-2)\).
  3. B: \(R_{(180^{\circ},O)}(1,3) = (-1,-3)\).
  4. C: \(R_{(180^{\circ},O)}(-2,3) = (2,-3)\).

Example 2.3 - Rotation by 270° - Counter-Clockwise

Rotate \(\triangle ABC\) counter-clockwise by \(270^{\circ}\) about the origin.

  1. Remember \(r_{(270^{\circ},O)}(x,y) = (y,-x)\).
  2. A: \(r_{(270^{\circ},O)}(1,2) = (2,-1)\).
  3. B: \(r_{(270^{\circ},O)}(1,3) = (3,1)\).
  4. C: \(r_{(270^{\circ},O)}(-2,3) = (3,2)\).

3. Rotations Around the Origin - Clockwise
Instruction Examples Video Self-Check

If you have to rotate a figure clockwise, you can use the same rules as the counter-clockwise with some modification.

The important concept you need to understand about clockwise rotation around the origin is the following:

Counter-Clockwise Rotation Clockwise Rotation
\(90^{\circ}\) \(270^{\circ}\)
\(180^{\circ}\) \(180^{\circ}\)
\(270^{\circ}\) \(90^{\circ}\)
\(360^{\circ}\) \(360^{\circ}\)

When a coordinate or figure is rotated clockwise around the origin, then the following is true:

  1. \(R_{(90^{\circ},O)} \rightarrow R_{(270^{\circ},O)}(x,y) = (y,-x)\)
  2. \(R_{(180^{\circ},O)}(x,y) = (-x,-y)\)
  3. \(R_{(270^{\circ},O)} \rightarrow R_{(90^{\circ},O)}(x,y) = (-y,x)\)
  4. \(R_{(360^{\circ},O)}(x,y) = (x,y)\)

Example 3.1 - Rotation by 90° - Clockwise

Rotate \(\triangle ABC\) clockwise by \(90^{\circ}\) about the origin.

  1. Remember a rotation of \(90^{\circ}\) clockwise is the same as a \(270^{\circ}\) rotation counter-clockwise. Therefore, you should use the formula for \(270^{\circ}\).
  2. \(R_{(270^{\circ},O)}(x,y) = (y,-x)\).
  3. A: \(R_{(270^{\circ},O)}(1,2) = (2,-1)\).
  4. B: \(R_{(270^{\circ},O)}(1,3) = (3,-1)\).
  5. C: \(R_{(270^{\circ},O)}(-2,3) = (3,2)\).

Example 3.2 - Rotation by 270° - Clockwise

Rotate \(\triangle ABC\) clockwise by \(270^{\circ}\) about the origin.

  1. Remember a rotation of \(270^{\circ}\) clockwise is the same as a \(90^{\circ}\) rotation counter-clockwise. Therefore, you should use the formula for \(90^{\circ}\).
  2. \(R_{(90^{\circ},O)}(x,y) = (-y,x)\).
  3. A: \(R_{(90^{\circ},O)}(1,2) = (-2,1)\).
  4. B: \(R_{(90^{\circ},O)}(1,3) = (-3,1)\).
  5. C: \(R_{(90^{\circ},O)}(-2,3) = (-3,-2)\).
