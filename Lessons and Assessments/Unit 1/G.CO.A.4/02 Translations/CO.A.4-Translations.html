1. Translation
Instruction Examples Video Self-Check

Translation is a type of rigid transformation in which a figure slides or shifts on a plane. It does not change its shape or size.

Each point on the pre-image figure moves the same distance in the same direction to create the image figure.

In the image to the right, the \(BAT\) has been translated twelve units up to create \(B'A'T'\).

Pre-image \(BAT\) and image \(B'A'T'\) on the same graph.

Describing Translations using Translation Notation

On the coordinate plane, the \(x\) axis travels side to side. Moving right, which is positive, is written \(x\) plus the number of units. Moving left, which is negative, is written \(x\) minus the number of units.

The same rules apply for moving up and down, which is the \(y\) axis. Moving up is \(y\) plus the number of units. Moving down is \(y\) minus the number of units.

Here, \(\triangle ABC\) has been translated by moving each point to the left \(3\) units and up \(4\) units. We can write this translation as \((x, y) \longrightarrow (x-3, y+4)\). This shows that for every single point, the \(x\) moves to the left \(3\) units \((- 3)\), and the \(y\) moves up \(4\) units \((+ 4)\).

Translations can also use a vector which provides both the direction and distance of the change in movement. Above, the vector notation for left \(3\) units and up \(4\) units is \(\left\langle -3,4\right\rangle\). Vector notation is shorter but is not as easy to use as translation notation when finding translated points.

Translation with Change in \(x\) and \(y\)

Describe Translations

Write the change in the pre-image to the image using translation notation and vector notation. Translation is written as: \(\left(x,\ y\right)\longrightarrow \left(x\ \pm \ a\ ,\ y\pm b\right)\) where a and b are the changed values. Vector is written as: \(\left\langle a,b\right\rangle\) where \(a\) and \(b\) are the changed values.

Example 1.1

Here, the pre-image translates seven left and three down. In translation notation, this is written \(\left(x,\ y\right)\longrightarrow \left(x\ -7\ ,\ y-3\right)\). Both values are negative because a move left and down are both moves in the negative direction. The vector notation is written \(\left\langle -7,-3\right\rangle\).

Example 1.2

Sometimes, a figure will move only either vertically or horizontally instead of both. Here, ∆\(CAP\) has been translated by moving each point right by five units but zero units up or down. The translation notation is written \(\left(x,\ y\right)\longrightarrow \left(x\ +5\ ,\ y\right)\). Notice the \(x\) moved right five units \((+5)\) and the \(y\) remains as \(y\) alone because it has not changed. Do not use zero in the translation notation. The vector notation is written \(\left\langle 5,0\right\rangle\). A zero must be used for vector notation.

2. Creating Translations
Instruction Examples Video Self-Check

You can create translations by using the given coordinates and the provided translation.

Here, translate \(R\left(3,\ -4\right)\) left five units and up one unit.
  • To find the image, first write the translation notation for left five units and up one unit: \(\left(x,\ y\right)\longrightarrow \left(x\ -5,\ y\ +1\right)\).
  • Then combine the coordinates of the original point with the translation.
  • The image of \(R\left(3,\ -4\right)\) with translation \(\left(x,\ y\right)\longrightarrow \left(x\ -5,\ y\ +1\right)\), can be found by \(R\left(3,\ -4\right)\longrightarrow \left(3\ -5,\ -4\ +1\right)=\ R'\left(-2,\ -3\right)\).

    • Translating a Coordinate

    Example 2.1

    Translate \(G\left(-4,\ 3\right)\) by the translation \(G\left(x,\ y\right)\longrightarrow \left(x\ +5,\ y\ +6\right)\).

    Solution:
    Substitute the given \(\left(x,\ y\right)\) values for point \(G\) to find \(G’: G\left(-4,\ 3\right)\longrightarrow \left(-4\ +5,\ 3\ +6\right)=G'\left(-1,\ 9\right)\)

    Example 2.2

    \(Q\left(-1,\ 5\right)\) right three units and down two units.

    • First, write the translation notation for right three units and down two units: \(\left(x,\ y\right)\longrightarrow \left(x\ +3,\ y\ \ -2\right)\)
    • Then, plug in the coordinates for \(Q\) to find \(Q’: Q\left(-1,\ 5\right)\longrightarrow \left(-1\ +3,\ 5\ \ -2\right)=\ Q'\left(2,\ 3\right)\)

    Example 2.3 A

    Translate quadrilateral \(QRST\) by \(\left\langle -9,-1\right\rangle\). Then give the coordinates of each image point.

    • First, convert the vector notation for \(\left\langle -9,-1\right\rangle\) into translation notation: \(\left(x,\ y\right)\longrightarrow\left(x\ \ -\ 9,\ y\ -\ 1\right)\).
    • Then take each of the coordinates and enter them into the new notation to solve.
