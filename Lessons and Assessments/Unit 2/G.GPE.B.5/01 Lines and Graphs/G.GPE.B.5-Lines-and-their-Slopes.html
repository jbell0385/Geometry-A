Definition of a Line

A line is a collection of infinite points that travel forever in two opposite directions.

  1. A line cannot be curved.
  2. A line goes on forever, which is indicated by the arrows on the ends of the line.
  3. There is an infinite number of points on a line.

Key features of a Line

The two most important features of a line:

  1. Y-intercept
  2. Slope
Y-Intercept

The y-intercept is where the line intersects the y-axis (vertical axis).

Slope

The slopes indicates the direction of a line by finding its rate of change between two points on the line.

Finding the Y-Intercept and Slope

There are several ways to find the y-intercept and slope. The example tab will explore these different methods.

Example 1.1: Two Slope Methods on a Graphed Line

Method 1

When given the graph of a line, calculate the slope by using any two points on the line that lie on the intersection of the grid lines and use the following ratio:

\(slope=\frac{rise}{run}\)

The rise is the vertical change between the points and the run is the horizontal change between the points. Count the number of boxes between each point vertically and divide by the number horizontally. Always simplify fraction answers.

Method 2

Slope is also found by finding the difference (subtraction) in y values and then dividing it by the difference (subtraction) in x values with this formula:

\(slope=\frac{change\ in\ y\ values}{change\ in\ x\ values}=\frac{y_2-y_1}{x_2-y_2}\)

Find the slope of the following line using both methods above.

Solution: Example 1.1

The slope = 3 using both methods. To start, choose two points that fall on the intersection of the grid lines. For method 1, count the number of boxes between points. Divide the vertical number of boxes by the horizontal number of boxes. For method 2, find and label the coordinates for each point. Then plug them into the slope formula.

Example 1.2: Zero Slope

Lines that travel horizontally (side to side) have a slope of zero because by the slope formula:

\(slope=\frac{change\ in\ y\ values}{change\ in\ x\ values}=\frac{y_2-y_1}{x_2-y_2}\),

the change in the \(y\) values will always be zero because the \(y\) coordinate stays the same. This makes the final slope ratio always zero.

Find the slope and the y-intercept below.

Solution: Example 1.2

Example 1.3: Undefined Slope

Lines that travel vertically (up and down) have a slope that is undefined or that does not exist because by the slope formula:

\(slope=\frac{change\ in\ y\ values}{change\ in\ x\ values}=\frac{y_2-y_1}{x_2-y_2}\),

the change in the \(x\) values will always be zero because the \(x\) coordinate stays the same. The \(x\) values lie in the denominator of the fraction and when any number is divided by zero, the result is no solution. On the coordinate plane, a line that has no solution for its slope is has an undefined slope.

Find the slope and the y-intercept below.

Solution: Example 1.3

Since the line does not intersect the y-axis, there is no \(y\)-intercept. This does not mean that the \(y\)-intercept is zero; there is no value where the line will cross the vertical axis.

Find Slope and Y-intercept Using Slope-Intercept Form

Slope-Intercept Form is an equation used to describe a line where the slope is represented as \(m\) and the \(y\)-intercept is represented as \(b\). The \(m\) and \(b\) are written as values and \(x\) and \(y\) are written as variables in the final equation. Altogether, the equation is:

Graph Equations Using Slope and Y-intercept

Slope-Intercept Form also creates an easy way to graph the line created from the equation. Here, the line is graphed by using both the slope and \(y\)-intercept with the following values.

Graph the equation of a line with a slope of -2 and a \(y\)-intercept of 3.

Steps to Graph

Step 1: Graph the \(y\)-intercept point. Remember, this is where the line crosses the \(y\)-axis.

Step 2: Use the slope to determine the location and plot another point. To graph easily, slopes should be in fraction form. If the slope value is a whole number, remember that you can always divide by 1 to create a fraction.

Since Slope is the \(\frac{change\ in\ y\ values}{change\ in\ x\ values}\), the numerator is how far to move on the \(y\)-axis and the denominator is how far to move on the \(x\)-axis.

Step 3: Connect the points

3. Create Slope-Intercept Form Using Two Points
Instruction Examples Video Self-Check

Now you will combine the two previous ideas: finding the slope and \(y\)-intercept of a line without graphing and using them to create an equation in slope-intercept form.

When a graph is not provided, it is helpful to use the previous strategy in the first learning block to solve for the slope. When given two ordered pairs which represent points on the line, always label the values of the given two points in order to decide which value is which variable. Label as follows:

\(\left(x_1,\ y_1\right)\) and \(\left(x_2,\ y_2\right)\)

Next, substitute the values into the slope equation in order to solve for the slope. The slope equation from before is:

\(slope=\frac{change\ in\ y\ values}{change\ in\ x\ values}=\frac{y_2-y_1}{x_2-y_2}\)

Last, the \(y\)-intercept, represented as the \(b\), is the final value needed to write an equation in slope-intercept form. To find this, substitute either ordered pair in for the \(x\) and \(y\) values in \(y=mx+b\) and solve for \(b\). Click on the examples tab to practice creating slope-intercept form.

Example 3.2

Solution:

To write the equation, first find the slope by labeling the points and using the slope formula. After, use that value and choose either ordered pair to represent the \(x\) and \(y\) values in \(y=mx+b\) and solve for \(b\).
