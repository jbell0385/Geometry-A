1. Proportional Relationships
Instruction Examples Video Self-Check

Recall that a ratio is a comparison of two numbers. If there are \(5\) boys and \(3\) girls in a class, then the ratio of boys to girls is \(5:3\) or \(5/3\). Remember, ratios must always be simplified.

A proportion is a comparison of two equal ratios. \(\dfrac{4}{6} = \dfrac{10}{15}\) is a proportion because both ratios can be simplified to \(\dfrac{2}{3}\).

Example 1.1

Many times, there is a missing value in a proportion. Consider the proportion: \(\dfrac{4}{6} = \dfrac{x}{15}\)

In order to find the missing value of \(x\) you must use a method called cross multiplication.

  1. Cross multiply
  2. Set answers equal to each other
  3. Solve the equation for \(x\).

Example 1.2

Proportions can use a variable in multiple locations and include extra terms. Follow the same steps as before.

  1. Cross multiply
  2. Set answers equal to each other
  3. Solve the equation for \(x\).
